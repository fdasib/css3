Intro

Arti

Generasi ke-3 dari spesifikasi CSS yang direkomendasikan oleh W3C

W3C : Organisasi yang membuat aturan-aturan terkait teknologi dan internet

Kenapa CSS3

  1. Membuat website menjadi lebih fleksibel
  2. Membuat mock-up dengan cepat tanpa bantuan bahasa scripting
  3. Mengurangi penggunaan gambar = kecepatan load halaman
  4. Penggunaan selector untuk menghindari penggunaan markup yang boros

Box Shadow

Fungsi

Untuk membuat bayangan pada element

Singkatan

IXYBC

Property wajib

x-offset : mengatur jarak bayangan pada horizontal/ atas-bawah

y-offset : mengatur jarak bayangan pada vertical/ kanan-kiri

blur : mengatur blur bayangan

color : mengatur warna bayangan

Example

box-shadow : [x-offset] [y-offset] [blur] [color]

box-shadow : 5px 5px 5px black

Transparasi Bayangan

box-shadow : 5px 5px 5px rgba(0, 0, 0, .75);

Property Spread

Spread >> sebaran

ukuran untuk bayangan

box-shadow : [x-offset] [y-offset] [blur] [spread] [color]

box-shadow : 30px 30px 5px 5px rgba(0, 0, 0, .75);

Penggunaan tepat spread

box-shadow : 0px 30px 10px -20px rgba(0, 0, 0, .75);

Property inset

Insert >> sisipan

membuat bayangan berada dikotaknya

box-shadow : [inset ][x-offset] [y-offset] [blur] [spread] [color]

box-shadow : inset 0px 30px 10px -20px rgba(0, 0, 0, .75);

Multiple Shadow

box-shadow : 5px 5px 5px rgba(255, 0, 0, .5),

-5px -5px 5px rgba(0, 0, 255, 5);

Multiple Border

box-shadow : 0px 0px 0px 5px red,

0px 0px 0px 10px green,

0px 0px 0px 15px yellow,

0px 0px 0px 20px orange;

Teks Shadow

Property

text-shadow : [x-offset] [y-offset] [blur] [color]

text-shadow : 5px 5px 10px green

Multiple Shadow

text-shadow : 5px 5px 10px green, -5px -5px 10px red

Linear Gradient

linear-gradient

bukan property CSS3, melainkan value untuk background-image

Sintaks sederhana

background-image : linear-gradient([warna-awal], [warna-akhir]);

background : linear-gradient([warna-awal], [warna-akhir]);

background-image : linear-gradient(red, green);

Multiple Color

background-image : linear-gradient([warna-1], [warna-2], ..., [warna-n]);

background-image : linear-gradient(red, yellow, green);

Sintaks Lengkap

background : linear-gradient([sudut] | [arah], [warna-awal], [warna-akhir]);

sudut

background-image : linear-gradient(90deg, red, green); << kanan ke kiri

background-image : linear-gradient(-90deg, red, green); << kiri ke kanan

background-image : linear-gradient(45deg, red, green); << diagonal

Arah

to bottom >> ke bawah
to top >> ke atas
to left >> ke kiri
to right >> ke kanan

to left top >> ke atas kiri
to left bottom >> ke bawah kiri
to right top >> ke atas kanan
to right bottom >> ke bawah kanan

background-image : linear-gradient(to right, red, green);

Color Stop

Memberikan bobot warna berbeda

background-image : linear-gradient(to right, red 63%, green 80%);

Radial Gradient

Sintaks sederhana

background : radial-gradient([warna-awal], [warna-akhir]);

background : radial-gradient(red, blue);

Multiple Color

background : radial-gradient([warna-1], [warna-2], ..., [warna-n]);

background : radial-gradient(red, green, blue);

Color Stop

background : radial-gradient(red 10%, green 20%, blue 80%);

Sintaks Lengkap

background : radial-gradient([tipe] [jangkauan] at [posisi-y] [posisi-x], [warna-1], [warna-2], ..., [warna-n])

Tipe

ellipse dan circle

background : radial-gradient(ellipse, red, blue)

background : radial-gradient(circle, red, blue)

Jangkauan

jarak dari pusat gradient ke sisi terluar-nya

closest-corner

jarak dari pusat gradient ke sudut terdekat element

background : radial-gradient(circle closest-corner, red, blue)

closest-side

jarak dari pusat gradient ke sisi terdekat element

background : radial-gradient(circle closest-side, red, blue)

farthest-corner

jarak dari pusat gradient ke sudut terjauh element

background : radial-gradient(circle farthest-corner, red, blue)

farthest-side

jarak dari pusat gradient ke sudut terjauh element

background : radial-gradient(circle farthest-side, red, blue)


menentukan sisi terdekat dan terjauh tergantung pada elementnya

Posisi

posisi-x

top

center

bottom

px


posisi-y

left

center

right

px


background : radial-gradient(circle farthest-side at top left, red, blue)

Filter

Fungsi

Untuk membuat efek pada gambar

Singkatan

B2 CSG SHI OD

B2 (Blur Brighness)
CSG (Contrast Saturate Grayscale)
SHI (Sepia Hue-rotate Invert)
OD (Opacity Drop-shadow)

Blur

Untuk Mengeblurkan gambar

filter : blur(30px); << px

Brighness

Untuk mengatur terang

filter : brightness(2); << angka atau % || 2 = 200%

contrast

Untuk membuat gambar agak gelap

filter : contrast(1.1); << angka atau % || 1.1 = 110%

Saturate

Untuk mengatur saturasi\warna gambar

filter : saturate(1.5); << angka atau % || 1.5 = 150%

Grayscale

Untuk mengatur abu-abu

filter : grayscale(1); << angka atau % || 1 = 100%

sepia

Untuk mengatur kuning/lama

filter : sepia(1.5); << angka atau % || 1.5 = 150%

hue-rotate

Untuk mengatur

filter : hue-rotate(100deg); << deg

Invert

Untuk mengatur negatif film

filter : invert(1); << angka atau % || 1 = 100%

Opacity

Untuk mengatur

filter : opacity(.5); << angka atau % || .5 = 50%

drop-shadow

Untuk mengatur

keuntugan : bisa untuk teks, box, gambar, dan png

filter : dop-shadow(5px 5px 5px black);

Menumpuk filter

filter : blur(5px) grayscale(100%)

perbedaan menggunakan opacity dan filter opacity

filter : yang bekerja hardwarenya

opacity dan box-shadow : yang bekerja browser

animation : [name] [durat ion] [timing-function] [delay] [iteration-count] [derection] [fill-mode] [play-state]

iteration-count

menentukan jumlah perulangan


<Angka>

infinite

direction

untuk menentukan arah


normal >> dari awal sampai akhir

reserved >> kebalikannya

alternate >> dari awal - akhir lalu dari akhir - awal

alternate-reserved >> dar akhir - awal lalu dari awal - akhir

fill-mode

menentukan akhir dari transtion


none

forwards >> tidak kembali posisi awal

backwards >> kebelakang

both >> kedua

play-state

menjalankan animasi


running >> berjalan

paused >> berhenti
